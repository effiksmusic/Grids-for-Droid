# Grids for Droid

This patch recreates some the functionality of Mutable Instruments' "Grids" module.

Copy `droid.ini` onto your Droid's SD card to start.

## The Droid patch

### Inputs

* `I1`: clock
* `I2`: reset
* `I5`: kick density (0v - 5v)
* `I6`: snare density (0v - 5v)
* `I7`: hihat density (0v - 5v)

### Outputs

* `O1`: kick accent
* `O5`: kick
* `O2`: snare accent
* `O6`: snare
* `O3`: hihat accent
* `O7`: hihat
